# Influencer Campaign Performance Tracker

##  Project Overview

The Influencer Campaign Performance Tracker is a data analytics project that analyzes influencer marketing campaigns using Python and Power BI. It helps evaluate campaign performance through key metrics, interactive dashboards, and data-driven insights.

---

##  Features

* Data cleaning and preprocessing using Python
* Campaign performance analysis
* KPI calculation and visualization
* Interactive Power BI dashboards
* Platform-wise performance comparison
* Revenue and ROI analysis
* Influencer performance tracking

---

##  Tech Stack

* Python
* Pandas
* NumPy
* Jupyter Notebook
* Power BI
* SQL
